# Alpine with Python 3.6, Tesseract OCR and OpenCV
This container contains the necessary to perform Optical Character Recognition.

- Python 3.6.1
- Tesseract 3.04.01
- OpenCV 3.3.0

# Getting container
Run `docker pull petronetto/alpine-opencv-python`
